# Nabeeha Midterms Gift

A React and Tailwind surprise site with a countdown, baby breath bouquet, and daily envelope letters.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:5173/`.

## GitHub Pages

This repo includes a GitHub Actions workflow that builds the Vite app and deploys the generated `dist` folder to GitHub Pages.

In the GitHub repo, go to **Settings > Pages** and set **Source** to **GitHub Actions**.
